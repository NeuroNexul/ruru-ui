## Usage

The `Block` is ready to use component that can be used to build your app faster.

Here is an example of how you can use the `Block` component:

<Tabs items={["npm", "pnpm", "yarn", "bun"]}>
<Tab value={"npm"}>
```bash
npx ruru-ui-cli@latest block [name]
```
</Tab>
<Tab value={"pnpm"}>
```bash
pnpm dlx ruru-ui-cli@latest block [name]
```
</Tab>
<Tab value={"yarn"}>
```bash
npx ruru-ui-cli@latest block [name]
```
</Tab>
<Tab value={"bun"}>
```bash
bunx --bun ruru-ui-cli@latest block [name]
```
</Tab>
</Tabs>
