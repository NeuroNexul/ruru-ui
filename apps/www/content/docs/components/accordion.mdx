## Usage

The `Accordion` component is a vertically stacked set of interactive headings that each contain a title, content, or both.

<Tabs items={["Preview", "Code"]}>

          <AccordionContent>Yes. It adheres to the WAI-ARIA design pattern.</AccordionContent>

## Props

| Prop | Description | Type | Default |
| ---- | ----------- | ---- | ------- |
| `asChild` | Render the accordion as a child component. | `boolean` | `false` |
| `type` | The type of accordion. | `"single" \| "multiple"` | `"multiple"` |
| `value` | The value of the accordion. | `string \| string[]` | `undefined \| []` |
| `defaultValue` | The default value of the accordion. | `string \| string[]` | `undefined \| []` |
| `onValueChange` | The callback function that is triggered when the value of the accordion changes. | `(value: string \| string[]) => void` | `undefined` |
| `collapsible` | Allow the accordion to be collapsible. | `boolean` | `false` |
| `disabled` | Disable the accordion. | `boolean` | `false` |
| `dir` | The direction of the accordion. | `"ltr" \| "rtl"` | `"ltr"` |
| `orientation` | The orientation of the accordion. | `"vertical" \| "horizontal"` | `"vertical"` |

| Data Attribute | Description | Values |
| -------------- | ----------- | ------ |
| `[data-orientation]` | The orientation of the accordion. | `"vertical" \| "horizontal"` |
