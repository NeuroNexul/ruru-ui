## Props

### Textarea

| Name          | Type                                                                                     | Default     | Description                                   |
| ------------- | ---------------------------------------------------------------------------------------- | ----------- | --------------------------------------------- |
| **className** | string                                                                                   | `""`        | Additional class names for the container.     |
| **label**     | string                                                                                   | `undefined` | Label for the textarea element.               |
| **error**     | string                                                                                   | `""`        | Error message to display below the textarea.  |
| **...props**  | **Omit\<React.InputHTMLAttributes\<HTMLTextAreaElement>, "children" "prefix" "suffix">** | `-`         | Any other props to be passed to the textarea. |
